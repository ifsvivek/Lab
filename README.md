## üîß How to Use This Repository

### Prerequisites

-   Python 3.x with pip
-   Java Development Kit (JDK) 8+
-   Node.js for web development
-   MATLAB (for .m files)
-   C compiler (GCC recommended)

### Running the Code

#### Python/ML Projects

```bash
cd Sem6/ML/Lab1
jupyter notebook Lab1.ipynb
# or
python ../PythonFiles/Lab1.py
```

#### Java Programs

```bash

java Lab1
```

#### C Programs

```bash
cd Sem2/C

./program
```

```bash

python -m http.server 8000  # For local server
```
